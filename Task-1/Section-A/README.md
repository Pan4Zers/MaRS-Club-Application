This project converts text input into Morse code and outputs it using a buzzer. 
Each character is translated into dots and dashes using a switch-case structure. 
The system demonstrates real-time encoding and synchronized output, simulating a basic communication system used in early telegraphy.

Experience & Learning:
I learned how to manage timing without delay(), which improved system responsiveness. Debugging taught me the importance of proper state handling and clean input processing. 
I also understood how hardware and software interact closely, especially in signal-based systems. 
This project helped me think more like an engineer rather than just writing code.
