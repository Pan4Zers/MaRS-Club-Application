This project converts text input into Morse code and outputs it using a buzzer and LED. \n Each character is translated into dots and dashes using a switch-case structure. \n The Arduino controls timing using millis(), allowing smooth, non-blocking signal generation. \n The system demonstrates real-time encoding and synchronized output, simulating a basic communication system used in early telegraphy.

\nExperience & Learning:
I learned how to manage timing without delay(), which improved system responsiveness. \nDebugging taught me the importance of proper state handling and clean input processing. \nI also understood how hardware and software interact closely, especially in signal-based systems. \nThis project helped me think more like an engineer rather than just writing code.
